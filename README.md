# JSON Tool Pro 🛠️

> **在线 JSON 工具箱 — 格式化、校验、对比、转换，纯前端，无需服务器。**

🌐 **在线使用：** [https://jsonprocess.app](https://jsonprocess.app)

![JSON Tool Pro 截图](https://jsonprocess.app/og-image.svg)

---

## ✨ 功能一览

### 基础功能
- **格式化** — 一键美化 JSON（缩进 2/4 可选）
- **压缩/最小化** — 去除多余空格，减小体积
- **校验** — 检查 JSON 语法错误，精确定位行号
- **转 CSV** — JSON 数组转表格格式，可用 Excel 打开
- **转义/反转义** — 在 JSON 字符串和可读文本之间转换

### 代码生成
- **Java POJO** — JSON 自动生成 Java 实体类代码
- **TypeScript 接口** — 自动生成 TypeScript 类型定义

### 高级工具
- **JSON Diff** — 对比两个 JSON 的差异，新增/删除/修改一目了然
- **JSONPath 查询** — 类似 XPath 的 JSON 数据定位查询
- **树形视图** — 树状结构浏览 JSON，更直观
- **JWT 解码** — 解码 JWT Token，查看 Header 和 Payload
- **转 YAML** — JSON 一键转换为 YAML 格式
- **JSON 生成器** — 生成随机测试数据

## 🏃 本地运行

```bash
# 克隆仓库
git clone https://github.com/675435543/json-tool.git

# 进入目录
cd json-tool

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
