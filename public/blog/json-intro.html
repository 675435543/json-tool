JSON 是什么？新手入门指南

2026 年 7 月 12 日 · 3 分钟阅读

什么是 JSON？

JSON（JavaScript Object Notation）是一种轻量级的数据交换格式。它基于 JavaScript 语法，但被几乎所有编程语言支持，包括 Java、Python、PHP、C# 等。

简单来说，JSON 就是用一种所有人都能看懂的方式来表示数据，方便不同系统之间传递信息。

一个最简单的 JSON 例子

{
  "name": "张三",
  "age": 25,
  "isStudent": false,
  "hobbies": ["读书", "编程", "跑步"]
}

这个 JSON 描述了一个人的基本信息：名字、年龄、是否为学生、以及兴趣爱好。是不是一眼就能看懂？

JSON 的六种数据类型

  • 字符串：用双引号包裹的文字，如 "hello"
  • 数字：整数或小数，如 42 或 3.14
  • 布尔值：true 或 false
  • 数组：用方括号包裹的值列表，如 [1, 2, 3]
  • 对象：用花括号包裹的键值对集合，如 {"key": "value"}
  • null：表示空值

JSON 的语法规则

  • 键名必须用双引号包裹（不能用单引号）
  • 字符串必须用双引号
  • 对象和数组可以任意嵌套
  • 最后一个键值对后面不能有逗号
  • JSON 文件通常以 .json 为扩展名

常见的 JSON 错误

新手常犯的错误包括：

  • 多了一个逗号（trailing comma）
  • 键名没用双引号
  • 字符串用了单引号
  • 布尔值写成了大写 True（应该小写 true）

JSON 有什么用途？

  • 前后端通信：网页和服务器之间传数据，目前最主流的方式
  • 配置文件：很多软件用 JSON 作为配置文件格式
  • 数据存储：NoSQL 数据库（如 MongoDB）直接使用 JSON 格式
  • API 数据交换：几乎所有开放 API 都返回 JSON 格式数据

为什么 JSON 这么流行？

相比 XML，JSON 更简洁、更易读。同样的一段数据，JSON 比 XML 少写一半字符，可读性也高得多。这也是为什么现在的 Web API 几乎都选择 JSON 而不是 XML。
